# crosswalk-ISAD
This repo used to save our works in crosswalk metadata from IPTC, PBCore, LoC Cataloque Map Collections to ISAD(G).

## Abstract 
Archival institutions manage diverse collections more than a single format or type. They often handle vast holdings comprising a wide variety of archives. Effective management of these collections requires describing the records using metadata standards. ISAD(G) is a widely recognized standard in the archival field used for archival description, and Access to Memory (AtoM) is an archival description application that utilizes it. However, given that ISAD(G) is a general metadata standard, questions arise regarding its compatibility in AtoM implementation for managing specific formats such as photographs, audiovisual recordings, and technical drawings especially since best practices for these types often involve complex and specialized metadata standards. This study aims to address this issue by developing a metadata crosswalk for several metadata used in describe archival collection(photographs, audiovisual recordings, and technical drawings). Conducted case study and action research methods, this study identify equivalencies across four metadata standards/practices (ISAD(G) in AtoM implementation, IPTC Photo Metadata, PBCore Metadata Dictionary for audiovisual, and Library of Congress catalogue with sample map collection. The resulting list of the crosswalk will serve as a reference for archival management and facilitate interoperability to describe multi-format archival materials using AtoM.
